# UV手电筒模块化原理图设计

## 1. LED模块原理图
```
[LED模块]
                热焊盘
                  ↓
            +----------+
            |  UV-LED  |
            |  365nm   |
            +----++----+
                 ||
            LED+ || LED-
                 ||
            红线 || 黑线
                 ↓↓
            To 恒流模块

器件参数：
- UV-LED: 365nm
- 正向电压: 3.7V
- 正向电流: 700mA
- 焊盘尺寸: 5×5mm
```

## 2. 充电管理模块原理图
```
[充电模块]
                    VCC(5V)
                      |
Type-C          C1(10μF)
VCC ---+--------+    |
       |        |    |
CC1 ---+     +----------+
       |     |          |
CC2 ---+     |  TP4056  |    R1(1.2K)
       |     |          |-------|<|--- LED1(红)
GND ---|     |          |
       |     |          |    R2(1.2K)
       |     |          |-------|<|--- LED2(绿)
       |     +----------+
       |          |
       |     R3(2.2K)
       |          |
       |          |    C2(10μF)
       |          |        |
       |     +-----------------+
       |     |     DW01-P     |
       |     |                 |
BAT+ --|     |                 |
       |     |            OD   |----+
BAT- --|     |            OC   |    |
       |     +-----------------+    |
       |            |              G1
       |            |              |
       |        FS8205A           G2
       |            |              |
       +------------+--------------+

器件清单：
1. Type-C: 16P母座
2. TP4056: SOP-8
3. DW01-P: SOT23-6
4. FS8205A: SOT23-6
5. LED: 0603红绿
6. C1,C2: 10μF/16V
7. R1,R2: 1.2KΩ
8. R3: 2.2KΩ
```

## 3. 升压模块原理图（新增）
```
[升压模块]
        VIN(1.7-2.8V)
            |
    C1(22μF)  L1(10μH)
            |     |
            |-----|-------> VOUT(3.7V)
MT3608 ----|     |
            |     |
         GND     D1(SS34)
            |     |
    C2(47μF)     |
            |     |
           GND    |

器件清单：
1. MT3608: SOT23-6
2. L1: 10μH/2A SMD
3. D1: SS34 SMB
4. C1: 22μF/16V
5. C2: 47μF/16V
6. R1,R2: 分压电阻(调3.7V)
```

## 4. 恒流驱动模块原理图
```
[恒流模块]
        VIN(3.7V)
            |
    C1(10μF)  L1(22μH)
            |     |
            |-----|-------> LED+
PT4115 ----|     |
            |     |
         GND     D1(SS34)
            |     |
    R1(0.2Ω)     |
            |     |
           LED-   |

器件清单：
1. PT4115: SOT23-6
2. L1: 22μH/1A SMD
3. D1: SS34 SMB
4. C1: 10μF/16V
5. R1: 0.2Ω/1%
```

## 5. 模块互连关系
```
[系统连接图]

Type-C → 充电模块 → 电池 → 升压模块 → 恒流模块 → LED模块
  5V       TP4056    1.7V    MT3608      PT4115     365nm
                     ↑         3.7V        700mA      3.7V
                     |
                  保护IC
```

## 6. 关键节点电压
```
1. 电源输入：5V DC
2. 电池电压：1.7-2.8V
3. 升压输出：3.7V
4. LED工作：3.7V@700mA
```

## 7. 保护电路参数
```
1. 过充保护：4.2V
2. 过放保护：2.5V
3. 过流保护：3A
4. 短路保护：<50ms
5. 充电限流：1A
```

## 8. 布线注意事项
```
1. 电源线
   - 主电源：20AWG
   - 信号线：22AWG
   - 走线宽度：>1.2mm

2. 信号线
   - PWM信号：0.25mm
   - 反馈信号：0.25mm
   - 远离电源线

3. 地线设计
   - 数字地分离
   - 模拟地独立
   - 单点接地
```
